## Dashboard with Mercury

**Notebook**

sf_crime_mercury.ipynb

**Instructions**

- Instal requirements.txt

- Run the notebook
- Open terminal 
  - go to the folder
  - activate the env
  - run the command _mercury watch sf_crime_mercury.ipynb_ 
- Open the indicated url to see the dashboard
